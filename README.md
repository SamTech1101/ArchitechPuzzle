## Quick publish (GitHub Pages)
1. Create a new **public** GitHub repo (example: `photo-puzzle`).
2. Upload these files to the **repo root**:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `multiclass_conference.jpg`
3. Repo **Settings ‚Üí Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Your link becomes:
   `https://YOUR-USERNAME.github.io/YOUR-REPO/`
